# Session 005 — App Shell

## What was built
- AppShell.jsx: flex layout wrapper, desktop sidebar + mobile bottom nav
- Sidebar.jsx: logo, nav links, profile card, sign out
- BottomNav.jsx: 6 main routes, active dot indicator, frosted glass bar
- Card.jsx and Button.jsx: base UI components with variants
- All page stubs updated with placeholder content
- Dashboard.jsx: time-based greeting using profile name

## Responsive behavior
- Mobile (< 768px): sidebar hidden, bottom nav visible, content full width
- Desktop (>= 768px): sidebar fixed 240px left, bottom nav hidden, content offset
